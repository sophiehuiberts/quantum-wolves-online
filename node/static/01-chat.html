Quantum werewolves

  • players set their name by `name yourname`
  • admin views roles with `roles`
  • admin sets roles like `roles Villager 2, Wolf 2`
  • admin starts a game using `start`
  • Start is in night 0
  • werewolf attacks are `attack yourname targetname`
  • seer viewings are `see yourname targetname`
  • admin moves to next night/day by `next`
  • to remove all players, use `new`
